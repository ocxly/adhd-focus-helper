# 🧠 OCXLY Neuro Lab v1.0

> Open-source tools for neurodivergent minds.

OCXLY Neuro Lab is a collection of practical browser-based tools designed to support focus, executive function, accessibility, sensory regulation, learning, and wellbeing.

## 🚀 Current Features

### Focus Tools
- ⏰ Pomodoro Pro
- 🧩 ADHD Task Chunker
- 🎉 Dopamine Menu

### Sensory Tools
- 🌧️ White Noise Generator
- 🟤 Brown Noise Generator
- 🌬️ Breathing Tool

### Wellbeing Tools
- 🧘 Grounding Tool (5-4-3-2-1)

### Accessibility
- 🌙 Theme Toggle
- 🔤 Lexend Typography
- 📱 Mobile-Friendly Design
